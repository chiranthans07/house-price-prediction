# House Price Prediction using Linear Regression

## Overview
This project predicts house prices using a Linear Regression machine learning model. The model is trained using features such as area, number of bedrooms, and bathrooms.

## Objective
To build a predictive model that estimates house prices based on housing features.

## Technologies Used
- Python
- Pandas
- Matplotlib
- Scikit-learn

## Dataset
The dataset contains information about houses including:
- Area
- Bedrooms
- Bathrooms
- Price

## Steps Performed
1. Data loading and exploration
2. Feature selection
3. Train-test split
4. Model training using Linear Regression
5. Prediction of house prices
6. Visualization of results

## Visualization
A scatter plot is used to compare actual vs predicted house prices.

## Key Insights
- House price increases with area.
- Number of bedrooms and bathrooms also influence price.
- Linear Regression effectively models this relationship.

## Conclusion
This project demonstrates how machine learning can be used to predict house prices and understand relationships between features and target variables.
